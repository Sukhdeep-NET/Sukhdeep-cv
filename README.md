# Sukhdeep Kambo — Portfolio Website

A modern, responsive portfolio website for **Sukhdeep Kambo**, Full Stack .NET Developer with 12+ years of experience in enterprise web applications, microservices architecture, and AI integration.

## Live Preview

[View Live](https://sukdevcv.vercel.app/)

## Features

- Light theme with floating gradient blobs
- Floating pill navigation bar
- Typewriter text animation
- Scroll reveal animations
- Responsive design (mobile, tablet, desktop)
- Contact form with interactive feedback
- Tech stack grid with devicon images
- Project cards with live links
- Experience cards with tech tags
- Download CV functionality

## Tech Stack Used

- HTML5
- CSS3 (Custom Properties, Grid, Flexbox, Animations)
- JavaScript (Vanilla)
- Google Fonts (Inter)
- Font Awesome Icons
- Devicon (Tech Stack Icons)

## Sections

1. **Hero** — Name, typewriter effect, Hire Me & Download CV buttons
2. **What I Offer** — 4 service cards (Backend, Database/CRM, API, AI)
3. **About Me** — Bio with download CV
4. **Projects** — 6 enterprise project cards with live links
5. **Experience** — 3 work experience cards with tech tags
6. **Tech Stack** — 12 technology icons grid
7. **Contact** — Message form + contact information
8. **Footer** — Credits

## Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/deeksha7sharma/sukdevcv.git
   ```
2. Open `index.html` in a browser or serve via any web server.
